# a_bear
### Install requirements    
  1.  Run in terminal

    pip3 install -r requirements.txt

### Open allure report
1. To install allure run in terminal: 
    
        'brew install allure' 
    
2. To open allure report run in terminal : 

        allure serve "$PWD/report"
        
3. Enjoy
