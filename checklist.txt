/info
Пользователь может получить описание по принципам работы сервиса

/bear - create
Пользователь может создать медведя с типом POLAR
Пользователь может создать медведя с типом BROWN
Пользователь может создать медведя с типом BLACK
Пользователь может создать медведя с типом GUMMY
Пользователь может создать нескольких медведей с корректным типом (здесь и далее корректный тип это POLAR, BROWN, BLACK или GUMMY)
Пользователь не может создать медведя без типа
Пользователь не может создать медведя с типами NaN/None/null
Пользователь не может создать медведя с некорректным и очень длинным типом
Пользователь не может создать медведя с некорректным и цифровым типом
Пользователь не может создать медведя с некорректным типом, который является исполняемым кодом

Пользователь может создать медведя с целым возрастом
Пользователь может создать медведя с очень большим возрастом
Пользователь может создать медведя с не целым возрастом
Пользователь может создать медведя с возрастом 0 лет (новорожденный медведь тоже медведь)
Пользователь не может создать медведя с отрицательным возрастом (дискуссионный вопрос, я бы созвал митинг по этому поводу:)
Пользователь не может создать медведя с не цифровым возрастом
Пользователь не может создать медведя с не цифровым и очень длинным возрастом
Пользователь не может создать медведя с не цифровым возрастом, который является исполняемым кодом
Пользователь не может создать медведя с существующим id

Пользователь может создать медведя с латинским именем
Пользователь может создать медведя с кириллистическим именем
Пользователь может создать медведя с именами на всех языках
Пользователь может создать медведя с очень длинным именем
Пользователь может создать медведя с именем из одного символа
Пользователь может создать медведя с именем, включающим в себя пробелы
Пользователь может создать медведя с именем, начинающимся с пробела
Пользователь может создать медведя с именем, заканчивающимся на пробел
Пользователь может создать медведя с именем, которое является исполняемым кодом
Пользователь может создать медведя с именами NaN/None/null
Пользователь не может создать медведя с пустым именем
Пользователь не может создать медведя с именем, состоящим из одного пробела
Пользователь не может создать медведя с именем, состоящим из нескольких пробелов

/bear/:id - delete specific bear
Пользователь может удалить единственного медведя
Пользователь может удалить одного медведя, но остальные должны остаться
Пользователь может удалить несколько медведей подряд
Пользователь может удалить медведя с именами NaN/None/null
Пользователь не может удалить несуществующего медведя

/bear - delete all bears
Пользователь может удалить нескольких медведей подряд
Пользователь может удалить всех медведей, если количество медведей равно 0
Пользователь может удалить всех медведей, если количество медведей равно 1
Пользователь может удалить всех медведей, если количество медведей равно 2
Пользователь может удалить всех медведей, если количество медведей много больше 1
Пользователь может удалить всех медведей, если количество медведей очень велико

GET			/bear - read all bears
Пользователь может получить информацию о всех медведях, если медведь 1
Пользователь может получить информацию о всех медведях, если медведей 2
Пользователь может получить информацию о всех медведях, если их медведей 0
Пользователь может получить информацию о всех медведях, если их медведей очень много


GET			/bear/:id - read specific bear
Пользователь может получить информацию о конкретном существующем медведе, если он единственный
Пользователь может получить информацию о конкретном существующем медведе, если медведей 2
Пользователь может получить информацию о конкретном существующем медведе, если медведей очень много
Пользователь может получить информацию о конкретном существующем медведе с любым корректным типом
Пользователь не может получить информацию о конкретном медведе, если он не существует
Пользователь не может получить информацию о конкретном медведе, если медведь существовал, но был удалён
Пользователь не может получить информацию о конкретном медведе, если медведь существовал, был удалён и вместо него создали такого же (но с другим id)
Пользователь не может вместо id медведя использовать NaN/None/null
Пользователь не может вместо id медведя использовать латинскую строку
Пользователь не может вместо id медведя использовать кириллистическую строку
Пользователь не может вместо id медведя использовать очень длинную строку
Пользователь не может вместо id медведя использовать строку из одного символа
Пользователь не может вместо id медведя использовать строку, включающую в себя пробелы
Пользователь не может вместо id медведя использовать строку, начинающуюся с пробела
Пользователь не может вместо id медведя использовать строку, заканчивающуюся на пробел
Пользователь не может вместо id медведя использовать строку, которая является исполняемым кодом
Пользователь не может вместо id медведя использовать строку, пустую строку
